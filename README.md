# SparkRSS
### A news reader designed for busy people

SparkRSS is an open-source, AI-powered RSS reader. Spark is lightweight and easy-to-use, with a responsive user interface and an easy way to import and export your list of feeds. SparkRSS can also be used alongside Bolt, the Ollama-powered AI backend that enables in-app summarization of the posts in your feed.

### Usage

Import this repository into the latest version of Android Studio, and hit Run when project import completes.
